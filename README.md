# microth - 基于 ESP32-C3 与 Python 的轻量化温湿度远程监控系统

本项目是一个基于高性价比 ESP32-C3 芯片与 Python FastAPI 异步后端的轻量化温湿度远程监控系统。支持内网多设备自动上报、SQLite 异步时序存储、飞书告警防轰炸通知以及自适应 Web 历史趋势图查看。

## 项目结构
```
microth/
├── firmware/         # ESP32-C3 采集端固件代码
│   └── README.md
├── server/           # Python 后端程序及可视化前端
│   └── README.md
└── 基于 ESP32-C3 与 Python 的轻量化温湿度远程监控系统 v2.0.md  # 详细设计方案 v2.0
```

## 核心设计特性 (v2.0)
1. **API 安全认证**：客户端采用 `X-API-Key` 密钥校验，防止数据篡改与恶意上报。
2. **异步高并发处理**：后端基于 FastAPI + `aiosqlite` 异步 SQLite 驱动，绝不阻塞服务事件循环。
3. **告警持久化与防轰炸**：飞书机器人告警状态及上次告警时间戳存入数据库，具备 30 分钟冷却期且不怕程序重启。
4. **数据自动老化**：内置定时任务，每日凌晨自动清理 90 天前的历史时序数据。
5. **部署支持**：支持 Systemd 服务管理和 Docker Compose 容器化一键部署。

# ESP32-C3 温湿度采集端固件

本目录存放基于 ESP32-C3 和 SHT40 传感器的温湿度采集端硬件固件代码。

## 硬件连接
- SHT40 VCC -> 3.3V
- SHT40 GND -> GND
- SHT40 SCL -> GPIO 9
- SHT40 SDA -> GPIO 8

## 功能特点
1. **网络自愈**：断网非阻塞重连，断连超5分钟自动重启网络协议栈。
2. **硬件看门狗**：开启 15 秒看门狗防止程序锁死。
3. **数据上报**：每 60 秒通过 HTTP POST 将 JSON 数据上报给后端，包含 `device_id`。
4. **安全认证**：在请求头中携带 `X-API-Key` 密钥认证。
